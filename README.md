# getStatus

    A python tool (async) to get Status and display
    
 ## Install 
 
    git clone https://github.com/0xGodson/getStatus
    cd getStatus


## Usage:

    ./getStatus <list_of_urls>

## Example:
 
    ./getStatus urls.txt 
    
## To save the Output

    ./getStatus urls.txt > status.txt

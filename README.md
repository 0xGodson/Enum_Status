# Enum_Status

    A python tool to get status and store the list of urls with the same status code. 
    
 ## Install 
 
    git clone https://github.com/0xGodson/Enum_Status
    cd Enum_Status
    chmod +x install.sh
    ./install.sh

## Usage:

    ./emumstatus <project_name> <list_of_urls>

## Example:
 
    ./enumstatus google urls.txt

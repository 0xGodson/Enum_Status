# Enum_Status

    A python tool to get status and store the list of urls with the same status code. 
    
 ## Install 
 
    git clone https://github.com/0xGodson/getStatus
    cd getStatus


## Usage:

    ./getStatus <list_of_urls>

## Example:
 
    ./getStatus urls.txt 
    
## To save the Output

    ./getStatus urls.txt > status.txt
